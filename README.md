🛒 MERN E-Commerce Store

A full-stack E-Commerce web application built using the MERN stack. The platform allows users to browse products, manage carts, place orders, and enables administrators to manage products, categories, users, and orders.

🚀 Features

---> User Features
- User Authentication & Authorization
- Browse Products
- Product Search & Filtering
- Add to Cart
- Favorites/Wishlist
- Checkout & Order Placement
- Order History

---> Admin Features
- Dashboard Analytics
- Product Management (CRUD)
- Category Management
- User Management
- Order Management
- Sales Statistics & Charts

🛠️ Tech Stack

---> Frontend
- React.js
- Redux Toolkit
- RTK Query
- Tailwind CSS
- React Router

---> Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer

---> Dashboard Features
- Total Sales
- Total Orders
- Total Customers
- Sales Trend Visualization

---> 🔐 Authentication
- JWT Based Authentication
- Protected Routes
- Admin Authorization

---> 📷 Image Upload
- Product Image Upload using Multer

---> ⚡ Installation

```bash
git clone <https://github.com/ABHINAVNAUTIYAL079/e-commerce-store>
cd project
npm install
npm run dev
```
